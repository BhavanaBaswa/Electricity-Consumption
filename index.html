About the Project

This project explores electricity consumption trends across India using historical electricity consumption data from January 2019 to December 2020.

The dataset was stored in SQL Server, analyzed using SQL queries, visualized using Tableau Public, and embedded into a Flask web application.

Project Objectives

  • Analyze electricity consumption trends (2019–2020)
  • Study the impact of the COVID-19 lockdown
  • Compare state-wise and region-wise electricity usage
  • Build interactive dashboards and stories using Tableau
  • Store and analyze data using SQL Server
  • Integrate Tableau dashboards with a Flask web application

Project Summary

A brief overview of the Electricity Consumption Analysis project and its outcomes.

Electricity Consumption Analysis (2019–2020)

The Electricity Consumption Analysis project provides a comprehensive analysis of electricity usage across Indian states and regions from January 2019 to December 2020. The project focuses on identifying consumption trends before, during, and after the COVID-19 lockdown through interactive dashboards and data visualizations.

The dataset was stored and managed using Microsoft SQL Server, analyzed using SQL queries, and connected to Tableau for creating interactive dashboards, maps, charts, filters, and stories. Finally, the Tableau dashboards were published to Tableau Public and integrated into a Flask web application to provide an interactive and user-friendly analytical platform.
